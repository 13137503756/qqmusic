# music

参考文献 https://jsososo.github.io/QQMusicApi/#/

使用者先在vue.config.js中的host和post配置自己电脑的ip和端口号  端口号随便写默认8080端口
然后再main.js中也配置下axios.defaults.baseURL的地址,要用http://加上自己的ip地址在加上端口号
